Atalhos

Aqui mostrarei atalhos que são úteis para agilizar ou facilitar a criação do seu site.

Recomendo usar o "Visual Studio Code" para desenvolver seus códigos, sendo uma alternativa melhor do que o Notepad++ (porém, também serão mostrados os atalhos para este programa)

Windows + V

Este comando serve para ativar a área de transferência do windows (é necessário ativa-lo antes de usar no Windows 10).

Com este comando, é possível copiar vários elementos ou palavras e deixá-las armazenadas para usar posteriormente.

Para colar, clique em algum lugar e aperte a bandeira do Windows + V, e então o menu aparecerá para que você escolha qual cópia você quer usar.

Obs: Esta área tem uma aparência um pouco diferente no Windows 10, porém possui a mesma função.

tips_and_updates

Se estiver usando algum computador/usuário novo, não se esqueça de ativar a área de transferência antes de usar! Faça Windows + V e clique em ativar na janela que aparecerá no canto da tela.

tips_and_updates

Quando este comando é utilizado, os papéis dos comandos justify-content e align-items se invertem, portanto o align-items agora irá alinhar horizontalmente, e o justify-content irá alinhar verticalmente.

Ctrl + F

Este comando funciona tanto no VSC (Visual Studio Code) quanto no notepad++

Serve para encontrar palavras no seu site, e (se necessário) substituí-las por outras.

Para usar, aperte Ctrl + F e digite a palavra que quer encontrar na barra de pesquisa. Para substituir por outra, clique na seta e digite a palavra que substituirá a outra no campo abaixo.

Clique no botão da esquerda para substituir um de cada vez, e o botão da direita para substituir todos de uma vez.

No notepad++ o comando é o mesmo, porém para substituir, vá para a aba replace e siga os mesmos passos:

Para substituir todos, clique em "Replace all"

Outros Atalhos:

Shift + Alt + Seta para baixo Duplica uma linha (VSC).

Ctrl + D Duplica uma linha (Notepad++).

Shift + Scroll Para navegar horizontalmente pelo código.

Ctrl + A Seleciona tudo.

Ctrl + Shift + T Abre denovo a última aba que foi fechada.

Ctrl + S Salva o arquivo.

Ctrl + Shift + S Salva o arquivo com outro nome.

Ctrl + X Copia um elemento e o apaga logo em seguida (recortar).
