Links

Lista de links de sites que podem auxiliar o desenvolvimento de um site.

Ícones

Sites relacionados ao uso de ícones.

Favicon

Para criar e customizar o ícone que aparece ao lado do título do site no navegador, a partir de um emoji, texto ou imagem.

Acessar o site

Ion-Icons

Coleção de ícones gratuitos e de fácil implementação, contando também com logos de empresas e sites, sendo esta a vantagem deste site.

Acessar o site

Google Icons

Coleção de ícones do Google. É um pouco mais complicado de ser usado, mas a dificuldade é compensada pela grande variedade de ícones disponíveis.

Acessar o site

SVG Repo

Grande coleção de ícones gratuitos e sem licença. Eles podem ser baixados em png, em svg, ou podem ser implementados pelo código em svg.

Recomendo baixar os ícones em svg, pois neste formato eles não perdem a qualidade quando seu tamanho é modificado. Este talvez seja o melhor site se você estiver procurando ícones mais "artísticos".

Acessar o site

Imagens

Sites relacionados a imagens.

Removedor de Fundos

Dentre todas as opções de sites que removem o fundo de imagens, o melhor que encontrei até agora foi o site PhotoRoom Background Remover. Este é um site de edição de imagens online, que possui uma função de remover fundos.

Ao enviar as imagens para o removedor, ele converte automaticamente as imagens em webp. para png, e preserva o tamanho da imagem original de graça, diferente de sites como o remove.bg que cobram por isso.

Acessar o site

Freepik

Imagens gratuitas em alta qualidade, possuindo uma grande variedade de imagens sobre qualquer tema. Tem imagens premium que precisam de pagamento, mas é possível encontrar a imagem que você precisa na versão gratuita.

É possível baixar 10 imagens por dia gratuitamente se você logar na sua conta do Google, e é necessário atribuir os créditos do autor caso você queira publicar seu site. Todas as imagens deste site foram obtidas no Freepik.

Acessar o site

SVG Backgrounds

Oferece alguns fundos em svg (ou png) gratuitos e customizáveis. Não é uma variedade tão grande, mas é perfeita para quem quer fundos que ocupem toda a página e com boa qualidade.

Acessar o site

Conversor de imagem

Converte imagens em qualquer extensão (avif, webp, etc) em png ou jpg.

Acessar o site

Compressor de imagens

Reduz o tamanho da imagem com o mínimo de perda de qualidade. Importante para reduzir o tempo de carregamento das imagens no site, assim como o tamanho dos arquivos.

Acessar o site

Geradores

Sites que geram determinados comandos automaticamente.

Gerador de CSS

Gerador de sombras, sombras de texto, bordas, curvatura das bordas, filtros, etc.

Existem diversos sites iguais a esse, porém este possui uma interface mais simples de se usar (para mim).

Acessar o site

Gerador de degradês

Cria efeitos de degradê customizáveis com duas ou mais cores e gera o código para você.

Acessar o site

Gerador de ondas (getwaves.io)

Gera imagens onduladas que podem ser usadas como separadores de seção em um site.

Este site em específico é bem simples e não conta com muitas possibilidades de customzação. Ele permite que você copie o código em svg, o que facilita a mudança rápida de cores da imagem.

Acessar o site

Gerador de separadores (shapedivider)

Gera imagens onduladas que podem ser usadas como separadores de seção em um site.

Possui mais opções de customização que o site anterior. Recomendo baixar o resultado como imagem em SVG neste caso.

Acessar o site

Gerador de ondas (svgwave.in)

Gera imagens onduladas que podem ser usadas como separadores de seção em um site.

Similar com os outros, porém este site oferece a animação das ondas (que só funciona no Google Chrome), e também um efeito de degradê.

Acessar o site

Gerador de separadores (coloreous)

Gera imagens onduladas que podem ser usadas como separadores de seção em um site.

Similar com os outros, porém este site oferece mais opções de desenho de separadores. Não é possível baixá-lo como imagem, porém é possível colar o código svg em algum editor de texto e salvá-lo como .svg.

Acessar o site

Gerador de bolhas (softr.io)

Cria uma bolha (blob) aleatória.

Acessar o site

Gerador de elementos SVG (fffuel.co)

Gera diversos tipos de elementos em SVG. Possui vários recursos interessantes para serem explorados.

Acessar o site

Gerador de imagens de fundo (bgjar)

Gera imagens de fundo customizáveis. É parecido com os outros, porém um dos estilos de fundo possui animação, que é bem simples, porém pode ser interessante para dar mais vida ao fundo do site.

Acessar o site
