W3Schools

Foi o primeiro site que usei e por onde aprendi a maioria das coisas que sei. Possui exemplos fáceis de entender e uma grande variedade de conteúdo.

Este site é totalmente em inglês, porém é possível traduzir o seu conteúdo clicando com o botão direito na página e clicando em "Traduzir para o português".

Acessar o site

Mozilla MDN

Talvez este não seja tão amigável com quem está começando, porém possui mais dconteúdo do que o W3schools. Disponibiliza absolutamente todos os comandos possíveis de CSS e seus conceitos, do simples ao avançado.

Sua vantagem é oferecer uma tradução para português, e mesmo oferecendo um conteúdo mais complexo, ainda oferece exemplos para facilitar a compreensão.

Acessar o site

StackOverflow

Ótimo site para tirar dúvidas mais específicas, onde pessoas com mais experiência te ajudarão. Este site possui uma versão em português, porém, é mais provável que você ache sua resposta mais facilmente em inglês.

É muito provável que alguém ja tenha tido a mesma dúvida que você antes, então não é necessário criar uma conta e perguntar você mesmo.

Acessar o site

Além desses sites, existem diversos outros espalhados pela internet. O Google terá a resposta para qualquer pergunta relacionada a CSS que você tiver.
